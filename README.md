# CraftRecipeUI

CraftRecipeUI 是一个易于上手的 Spigot 插件，可通过图形界面（UI）快速添加或移除 Minecraft 合成配方。  
CraftRecipeUI is a user-friendly Spigot plugin that lets you add or remove Minecraft crafting recipes through an intuitive UI.
